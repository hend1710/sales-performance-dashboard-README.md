Project Overview
A data-driven analysis project designed to evaluate customer satisfaction and service quality. By
processing raw feedback data, I created a system to track Net Promoter Score (NPS), guest
demographics, and service efficiency to improve the overall Customer Experience (CX).
Advanced Analytics & Insights
● Sentiment Analysis: Categorizing feedback into positive, neutral, and negative to identify pain points.
● Top Performing Categories: Identifying which services (e.g., Food, Cleanliness, Staff) received the
highest ratings.
● Profitable Guest Segments: Analyzing which guest types (Ref. Data) contribute most to the business
and their satisfaction levels.
● Peak Feedback Trends: Pinpointing the months with the highest engagement and service demand.
Technical Execution (SEO-Friendly)
● Data Integration: Linked "Guests Data" with "Feedback Data"
using advanced Excel functions (VLOOKUP/XLOOKUP) for a
unified view.
● Pivot Table Mastery: Generated dynamic summaries to compare
performance across different branches or service categories.
● Data Visualization: Created a Customer Feedback Dashboard
using slicers for real-time filtering by date, guest type, and rating.
The Business Value (The Result)
This dashboard transforms qualitative feedback into quantitative metrics,
allowing management to make Data-Driven Decisions that increase
customer retention by identifying and fixing service gaps
